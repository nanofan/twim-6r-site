# TWIM Site

Static site for Tranquil Wisdom Insight Meditation with:

- `index.html` for the main teaching page
- `timer.html` for the dedicated 6R timer
- `logo.svg` for the site mark

## Deploy

This is a plain static site with no build step.

- GitHub Pages: publish the repository root
- Netlify: publish the repository root
- Vercel: import as a static site

## Files

- `index.html` - homepage structure and content
- `timer.html` - dedicated timer page
- `styles.css` - shared layout and visual system
- `script.js` - homepage interactions
- `timer.js` - timer logic and bell sound
- `logo.svg` - brand mark
